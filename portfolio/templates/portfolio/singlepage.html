Projects

UCSD Parking Lots - Collecting and Analyze Large Data

This project at UC San Diego's Rady School of Management analyzes a decade-long dataset of parking lot usage across UCSD campuses, focusing on post-2018 data to examine the impact of COVID-19 on parking trends. Data was collected hourly from lots and structures and analyzed for patterns. Techniques used included web scraping, data transformation, Tableau for visualization, and Django for website development. The project aims to improve campus planning and resource allocation.

Carried out in the Fall quarter

Colega grupo: Grace Kim, Jeremy Edwards and Shridhar Rawat

Used Technologies: Tableau, Django, HTML, CSS, JavaScript

Mobility Dashboards - Final Course Work
