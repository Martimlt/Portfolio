Movies Recommendation and Rating Prediction - Web Mining & Recommender Systems

This project aims to create a movie recommendation system, focusing on user preferences and viewing history. Key steps include using a dataset with user ratings and movie preferences, and analyzing user behavior in film. The system will use machine learning for predicting user preferences and estimating ratings. The approach combines content-based filtering (cosine similarity scores) with advanced collaborative filtering methods like NMF and SVD. The most accurate model was used to recommend the top 10 movies to users.

UCSD Parking Lots - Collecting and Analyze Large Data

This project at UC San Diego's Rady School of Management analyzes a decade-long dataset of parking lot usage across UCSD campuses, focusing on post-2018 data to examine the impact of COVID-19 on parking trends.
