Projects

UCSD Parking Lots - Collecting and Analyze Large Data

This project at UC San Diego's Rady School of Management analyzes a decade-long dataset of parking lot usage across UCSD campuses, focusing on post-2018 data to examine the impact of COVID-19 on parking trends. Data was collected hourly from lots and structures and analyzed for patterns. Techniques used included web scraping, data transformation, Tableau for visualization, and Django for website development. The project aims to improve campus planning and resource allocation.

Carried out in the Fall quarter of the Master's degree

Team mates: Grace Kim, Jeremy Edwards and Shridhar Rawat

Used Technologies: Tableau, Django, HTML, CSS, JavaScript

Movies Recommendation and Rating Prediction - Web Mining & Recommender Systems

This project aims to create a movie recommendation system, focusing on user preferences and viewing history. Key steps include selecting a dataset rich in user ratings and movie preferences, and analyzing user behavior in film. The system will use machine learning for predicting user preferences and estimating ratings. The approach combines content-based filtering (cosine similarity scores) with advanced collaborative filtering methods like NMF and SVD. The most accurate model will be used to recommend the top 10 movies to users.

Carried out in the Fall quarter of the Master's degree

Used Technologies: Python
