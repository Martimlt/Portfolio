The system also monitors capacity, generates automated reports, triggers alerts, and streamlines procurement by placing cost-optimized orders.

Fake News Detection

This project developed a machine learning model to classify 44,000+ news articles as real or fake using NLP techniques. Preprocessing steps included text cleaning, tokenization, stopword removal, and lemmatization to prepare the data for feature extraction. A Logistic Regression model, trained with TF-IDF vectorization, achieved 98.3% accuracy, a 98% F1-score, a 1.84% false positive rate, and a 1.52% false negative rate. Cross-validation reinforced the model's robustness, while a word frequency analysis revealed key terms linked to misinformation patterns, offering insights into common fake news trends.

Technologies/Skills: Pyhton (Machine Learning, NLP)
